# UK Tax Calculator App

A Hala Digital Ltd prototype for UK Tax Calculator.

## What this includes

- Next.js app structure
- UK Self Assessment Calculator prototype for tax year 2025/26
- Lead capture demo
- SaaS positioning for accountancy practices
- Built-in pass/fail test checks
- Disclaimer and review warnings

## Important

This is a testing prototype and not final tax filing software.

## Deploy

1. Upload this project to GitHub.
2. Import the repository into Vercel.
3. Connect:
   - uktaxcalculator.co.uk
   - haladigital.co.uk
